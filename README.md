# Parqis
Digitalni sistem za upravljanje parkirnih mest, ki omogoča pregled razpoložljivosti, rezervacijo parkiranja ter avtomatsko preverjanje veljavnosti ob zaznavi registrske številke. Sistem povezuje uporabnike in upravljavce ter omogoča učinkovitejši nadzor, večjo pretočnost in boljšo izrabo parkirnih površin.
